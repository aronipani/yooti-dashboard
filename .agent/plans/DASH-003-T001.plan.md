## Implementation steps

1. Write failing tests for all 4 AC scenarios using @mock_aws
2. Create `get_current_metrics.py` handler:
   - Extract project_id from event["queryStringParameters"]
   - Validate project_id → 400 if missing/empty
   - Call repository.get_current_snapshot(project_id)
   - If None → 404 SPRINT_NOT_FOUND
   - If found → 200 with { project_id, sprint_num, kpis, insights, last_ingested_at }
   - Catch RepositoryError → 500 INTERNAL_ERROR
   - CORS headers in every response
3. boto3 client at module level
4. Test: assert GetItem is the operation used (not Query/Scan)
