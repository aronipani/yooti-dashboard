   - YOOTI_DASHBOARD_API_URL from os.environ
   - YOOTI_DASHBOARD_API_KEY from os.environ (empty for local)
   - Handle connection errors → clear message, raise
   - Handle non-2xx → clear message, raise
3. Write failing tests for aggregate-metrics.py main flow
4. Modify `aggregate-metrics.py`:
   - CLI entry point: `python pipeline/scripts/aggregate-metrics.py [--project-dir PATH] [--sprint NUM]`

6. Test: requests.post raises ConnectionError → exit code 1, clear message
