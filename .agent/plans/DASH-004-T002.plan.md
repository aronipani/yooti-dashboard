## Implementation steps

1. Write failing tests for AC-2, AC-3, AC-4
2. Create `get_sprint_detail.py` handler:
   - Extract sprint_num from event["pathParameters"]["sprint_num"]
   - Extract project_id from event["queryStringParameters"]
   - Validate sprint_num is positive integer → 400 INVALID_SPRINT_NUM
   - Validate project_id → 400 INVALID_PROJECT_ID
   - Call repository.get_sprint_summary(project_id, sprint_num)
   - If None → 404 SPRINT_NOT_FOUND
   - Call repository.get_sprint_stories(project_id, sprint_num)
   - Return 200 with { sprint_num, summary, phase_avg_hrs, gate_rejection_rates, escalation_breakdown, stories }
   - Catch RepositoryError → 500 INTERNAL_ERROR
   - CORS headers on all responses
3. boto3 client at module level
