## Implementation steps

1. Write failing tests for AC-3 and AC-4
2. Create `get_story_detail.py` handler:
   - Extract story_id from event["pathParameters"]["story_id"]
   - Extract project_id and sprint from event["queryStringParameters"]
   - Validate project_id → 400, validate sprint → 400
   - Call repository.get_story_detail(project_id, sprint, story_id)
   - If None → 404 STORY_NOT_FOUND
   - Call repository.get_phase_durations(project_id, sprint, story_id)
   - Assemble response: { story_id, sprint_num, type, status, cycle_time_hrs, phase_durations, gate_timestamps, agent_metrics, evidence, escalation_log }
   - Catch RepositoryError → 500
   - CORS headers on all responses
3. boto3 client at module level

## Constraints
